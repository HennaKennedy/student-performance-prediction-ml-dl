# Student Performance Prediction using ML & Deep Learning
This project predicts student academic performance using machine learning and deep learning techniques.

It compares multiple models to identify the most effective approach for structured educational data

## Problem Statement

Can prior academic performance, family factors, and behavioural attributes accurately predict student success?

## Models Used

- Logistic Regression
- Decision Tree
- Random Forest
- Neural Network
- Deep Neural Network
- K-Means Clustering (Unsupervised Learning)

## Results

| Model | Accuracy |
|------|--------|
| Logistic Regression | 87.34% |
| Decision Tree | 92.41% |
| Random Forest | 89.87% |
| DL Model 1 | 74.68% |
| DL Model 2 | 86.07% |

## Key Insights

- Prior grades are the strongest predictor
- Tree-based models outperform deep learning for tabular data
- Increasing model complexity does not guarantee better performance

## Dataset

UCI Student Performance Dataset:
https://archive.ics.uci.edu/ml/datasets/Student+Performance
